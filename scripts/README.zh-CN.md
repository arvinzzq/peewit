# Scripts

## Architecture Summary

这个目录拥有 repository maintenance scripts。
Scripts 提供支持 code review 和学习文档的项目检查。
Scripts 必须 deterministic，并且可以安全地在本地运行。

## File Inventory

| File | Role | Purpose |
| --- | --- | --- |
| `check-docs.ts` | Documentation checker | 验证 markdown links、bilingual heading counts、module guides 和 source headers。 |
